# Log Classification System

## Overview
This project builds an AI-powered log classification system that automatically categorizes system logs using Natural Language Processing (NLP), Regex-based parsing, and machine learning techniques.

The goal is to help identify important log events such as errors, security alerts, and HTTP responses from raw log data.

---

## Features
- Log dataset processing
- Regex-based log pattern extraction
- NLP-based feature extraction
- Machine learning log classification
- Support for multiple log sources
